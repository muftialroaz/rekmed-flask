# Rekmed Search Retrieval

This is search module for Rekmed-research by gamatutor.

You can find these repository in this [link](https://github.com/Rifqi2000/rekmed-research)

### How to run
```
python -m venv .venv
pip install -r requirements.txt
flask --app app run  
```
